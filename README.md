# 📊 E-Commerce Marketing Analytics

## 🚀 Overview
This project analyzes an e-commerce dataset to uncover insights into customer behavior, retention, revenue, and product trends.

---

## 🔍 Key Analysis

### 📈 Customer Acquisition
- Strong growth over time

### 🔁 Retention Analysis
- Very low retention after first purchase

### 💰 Revenue Analysis
- Revenue dominated by new customers

### 📦 Product Insights
- Few categories drive most revenue

### 💳 Payment Behavior
- Digital payments dominate

---

## 📊 Dashboard
Includes Power BI dashboard and Python analysis

---

## 🛠 Tools Used
- Python (Pandas, Matplotlib, Seaborn)

---

## 💡 Key Insight
The business is acquisition-heavy with low retention, indicating the need for customer engagement strategies.
